# Навоий вилояти — Боғланган ягона геоплатформа

Ер участкалари бўйича интерактив геоплатформа. Leaflet харита, туманлар бўйича KPI, қидирув, таклифлар тизими (Firebase Firestore орқали сақланади).

## Ишга тушириш (локал)

1. Файлларни очиш учун **`САЙТНИ_ИШГА_ТУШИРИШ.cmd`** ни икки марта босинг,
2. ёки `navoi_platform.html` ни браузерда локал сервер орқали очинг (масалан `python -m http.server 8080`).

## Асосий файллар

| Файл | Тавсиф |
|---|---|
| `navoi_platform.html` | Асосий саҳифа (UI, логика, Firebase интеграцияси) |
| `sections_data.js` | Бўлимлар бўйича маълумотлар |
| `navoi_geo.js` | Туманлар чегаралари (GeoJSON) |
| `changed_data.js`, `changed_extra.js` | Ўзгартирилган ер тоифалари |
| `platform_stats.js` | Умумий статистика |
| `leaflet.js`, `leaflet.css` | Харита кутубхонаси |
| `vercel.json` | Vercel деплой созламалари |

## Firebase

Таклифлар (`proposals` коллекцияси) **Firestore**да сақланади. Лойиҳа: `yer-loyiha`. Оффлайн режимда маълумотлар `localStorage`га тушади, интернет тикланганда автоматик синхронланади.

## Деплой

Сайт **Vercel** платформасига деплой қилинади. `vercel.json` орқали `/` → `navoi_platform.html`.
